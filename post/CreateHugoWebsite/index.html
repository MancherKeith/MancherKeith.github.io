Create Website By Hugo On Github.

Mancher Keith Jan 6, 2016

虽然自从大一上了C语言课就开始对编程开发之类的产生了极大的兴趣，之后也开始学习安卓，但是由于种种原因，总是疏于实践，一直都没有真正用代码写过什么东西。
今年临近毕业，大部分事情也结束了，终于下定决心开发点东西出来。注册了Github账号后也上传过一些代码，后来删来改去的也不剩啥了。
2015年就要过去了，总觉得不做点啥今年就白费了。于是，再次下决心整顿一下我的Github，搭建一个github主页，以后积极堆积小绿块。 废话说完，找教程去。。。。。PS：我并不是要写教程，只是记录一下自己的搭建过程，免得以后出问题了重建的时候还要再去找。

搜索一下Github blog，会出现一堆的教程，包括Jeklly，Hexo等等。以前我也只知道这两个，这次搜索的时候发现了Hugo这个意外的惊喜。

准备工作

需要有个Github账号，安装git工具，安装hugo工具。
- Github账号，注册一个就好。
- Git安装配置（网上教程很多） - hugo安装配置
到Hugo官方网站下载Hugo，图省事就直接下载编译过的。解压下载的安装包，里边有三个文件，把.exe文件放到自己觉得合适的目录下，然后把该目录路径添加到系统环境变量path中，其他的两个Markdown文件简单看看。
准备工作结束。
想对hugo有更多的了解，可以看看官方网站的docs，想看中文可以看看G-VAR’s Blog中的翻译
