Create Website By Hugo On Github.

教程集 搜索一下Github blog，会出现一堆的教程，包括Jeklly，Hexo等等。以前我也只知道这两个，这次搜索的时候发现了Hugo这个意外的惊喜。 记录一下看到的一些教程。 - Jeklly - 使用 GitHub, Jekyll打造自己的免费独立博客。——on_1y @CSDN blog - 如何搭建一个独立博客——简明 Github Pages与 jekyll 教程 - Hexo - Hugo - 使用hugo搭建个人博客站点。 - 使用Hugo搭建静态站点。—— Tony Bai - Hugo静态网站生成器中文教程。—— Nanshu Wang 这里只简单列举这么多，随便看了几个之后，最终选择使用Hugo。 准备工作 需要有个Github账号，安装git工具，安装hugo工具。 - Github账号，注册一个就好。 - Git安装配置（网上教程很多） - hugo安装配置 到Hugo官方网站下载Hugo，图省事就直接下载编译过的。解压下载的安装包，里边有三个文件，把.exe文件放到自己觉得合适的目录下，然后把该目录路径添加到系统环境变量path中，其他的两个Markdown文件简单看看。 准备工作结束。 想对hugo有更多的了解，可以看看官方网站的docs，想看中文可以看看G-VAR’s Blog中的翻译
